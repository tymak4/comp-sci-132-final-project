Tic Tac Toe
this project is a simple implementation of the classic Tic Tac Toe game. It allows 2 players to take turns marking spaces on a 3x3 grid, with the goal of getting three of their things in a row in any direction. 

features:
- 2 players are able to take turns placing their own markers
- input validation to prevent invalid inputs
- win/loss/draw detection

How it works:
the games uses a 2D list to represent the board. Players alternate turns, inputting positions. After each move, the program checks for a winning/draw condition. 

Getting Started: 
1. clone the repository
2. run the program
